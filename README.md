# เลขดี — เว็บแอปวิเคราะห์ "หวยลาว" จากสถิติ

เว็บแอปหน้าเดียว วิเคราะห์หวยลาวจากผลย้อนหลังจริง ทำงานในไฟล์ `index.html` ไฟล์เดียว
ไม่ต้องติดตั้งอะไร ไม่ต้องมี Node/npm

## เปิดใช้งานบนคอม
**ดับเบิลคลิก `index.html`** เปิดในเบราว์เซอร์ (Chrome / Edge) ได้เลย

## ปุ่ม "อัปเดตผลจริง"
กดปุ่ม **⟳ อัปเดตผลจริง** มุมขวาบน แอปจะ:
1. ดึงผลหวยลาวล่าสุดจาก [Sanook](https://www.sanook.com/news/archive/laolotto/) ผ่าน CORS proxy ฟรี (ยิงพร้อมกัน 4 ตัว เอาที่เร็วสุด — ปกติ 2–8 วิ)
2. เก็บลง **localStorage ของเบราว์เซอร์** → ประวัติสะสมขึ้นเรื่อยๆ ทุกครั้งที่กด
3. คำนวณสถิติใหม่ พร้อมป้าย **"ข้อมูลจริง N งวด"**

> **กดทุกวันเพื่อสะสมข้อมูล** — เว็บต้นทางโชว์ย้อนหลังแค่ไม่กี่งวด ครั้งแรกจึงได้น้อย
> ช่วงที่งวดยังน้อย ค่า % จะแกว่งมาก (เช่น 100%) เป็นเรื่องปกติทางสถิติ ยิ่งสะสมยิ่งแม่น

ถ้าขึ้น **"ดึงไม่สำเร็จ"** = พร็อกซีฟรีไม่ว่างชั่วคราว กดใหม่อีกครั้งสักครู่

## ❓ พรุ่งนี้จะกลับมาดูต้องทำยังไง

**ถ้าใช้บนคอมเครื่องเดิม:** เปิด `index.html` ซ้ำ → ข้อมูลที่เคยกดอัปเดตยังอยู่ (เก็บใน localStorage ของเบราว์เซอร์นั้น) แล้วกด "อัปเดตผลจริง" เพื่อดึงงวดใหม่เพิ่ม

**ถ้าอยากเปิดบนมือถือ / มีลิงก์เข้าได้ทุกที่:** ต้องเอาไฟล์ขึ้นโฮสต์ฟรีก่อน (ไฟล์ในคอมเปิดบนมือถือไม่ได้) วิธีที่ง่ายสุด:

### วิธี A — Netlify Drop (ลากวาง ได้ลิงก์ทันที)
1. เข้า https://app.netlify.com/drop
2. ลากไฟล์ `index.html` (หรือทั้งโฟลเดอร์) ไปวาง
3. ได้ลิงก์ เช่น `https://ชื่อสุ่ม.netlify.app` → เปิดบนมือถือได้เลย เพิ่มลงหน้าจอโฮมได้เหมือนแอป

### วิธี B — tiiny.host (ไม่ต้องสมัครก็ลองได้)
1. เข้า https://tiiny.host
2. อัปโหลด `index.html` → ได้ลิงก์ทันที

> **ข้อควรรู้:** ข้อมูลที่กดอัปเดตเก็บแยกตาม "อุปกรณ์/ลิงก์" (localStorage ต่อ origin)
> เปิดบนมือถือครั้งแรกจะว่าง ต้องกด "อัปเดตผลจริง" บนมือถือเองอีกที แล้วมันจะสะสมของมือถือต่อไป

## ปรับแต่ง / แก้เมื่อเว็บต้นทางเปลี่ยน
- ตัวแปลงผล (parser) อยู่ที่ `App.parse(html)` — ถ้า Sanook เปลี่ยนหน้าเว็บแล้วอ่านไม่ได้ ต้องแก้ regex ตรงนี้
- URL ต้นทาง: `App.SOURCE_URL`
- รายชื่อ proxy: ในเมธอด `App.fetchProxied`

### ใส่ประวัติย้อนหลังเองทั้งก้อน (ถ้ามีไฟล์/CSV)
```js
App._real = [ { ts: 1735689600000, top: 46, bottom: 54 }, ... ]; // เก่า → ใหม่
App.saveReal(); render();
```
- `ts` = มิลลิวินาที (`new Date('2025-01-01').getTime()`)
- `top` = 2 ตัวบน (0–99), `bottom` = 2 ตัวล่าง (0–99)
